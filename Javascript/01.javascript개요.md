## 개요

- 브라우저에서 실행된다
    - 따라서 모든 브라우저에서 동일한 실행결과가 나오지 않는다 (브라우저마다 다르다)
- 표준화된 자바스크립트
    - ECMASCript 6
    - 참고 페이지

    [JavaScript](https://developer.mozilla.org/ko/docs/Web/JavaScript)

- 프로그램 언어의 특징( ECMAScript 6 + ECMAScript 5 섞어서 학습 )
    - 데이터 종류
    - 변수
    - 함수
    - 연산자
    - 클래스
    - 이벤트
